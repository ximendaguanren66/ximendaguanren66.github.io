PHP 基础语法

1. if__els__

// <?php

// if(){ //括号中放条件
// 执行指令1
// }else{
// 执行指令2
// }
// ?>

2. switch__case__

// <? php

// switch () //括号中放条件
// {
// case 条件1:
// 执行指令1
// break;

// case 条件2:
// 执行指令2
// break;

// default:
// 默认指令
// }
// ?>
先判断条件，条件的返回值为条件1，则执行指令1;条件返回的值为条件2，则执行指令2;
条件的返回值既不是条件1也不是条件2，则执行默认指令。
break的作用是结束switch，使用 switch 语句可以避免冗长的 “if..else ；if..else”代码块。

3. while循

// <?php

// while(){ //括号中放条件
// 执行指令

// }
// ?>

首先判断某个条件是否符合，符合则执行任务，执行完毕任务，再判断条件是否满足要求，符合则重复执行此任务，否则结束任务。

4.php数据类型

//布 尔 型： True false (不区分大小写);
//整 型： 0，正负无穷到某个数值;
//浮点 型： 小数;
//字符 串： 带引号或者双引号的，以及>>>EOF EOF 之间的;
//数 组： [1,2,3,4,5] 索引数组，[“name”=>”25”] 关联数组;
//空 ： NULL （不区分大小写）;
//资源类型： STDIN,STDOUT。

5. 三元运算符

语法格式：
{expr1}?(expr2):(expr3)
?前边放条件，条件为真，返回冒号前面的值。为假返回冒号后面的值
应用实例：
<?php
$test = ‘大教程’;
//普通写法
$username = isset(test)?$test:’nobody’;

echo $username,PHP_EOL;

// PHP5.3+版本写法
$username = $test?:’nobody’;
echo $username,PHP_EOL;

?>

搭建的apache + php + mysql的环境（要求使用php5.6）

＃安装httpd
yum install httpd httpd-devel httpd-manual mod_ssl -y
＃启动apache
systemctl start httpd
＃设置apache开机启动
systemctl enable httpd
＃查看开放端口，检查有没有httpd服务，80和443端口是否开放
ss -antlp
配置防火墙，可以正常访问阿帕奇

firewall-cmd –zone = public –add-port = 80 / tcp
–permanent firewall-cmd –zone = public –add-port = 433 / tcp
–permanent firewall-cmd –reload

安装php5.6
rpm -Uvh https://mirror.webtatic.com/yum/el7/epel-release.rpm
rpm -Uvh https://mirror.webtatic.com/yum/el7/webtatic-release.rpm#overload
system. php
yum删除php-common -y

＃安装php5.6和php扩展
yum install -y php56w php56w-opcache php56w-xml php56w-mcrypt php56w-gd php56w-mysql php56w-intl php56w-mbstring

＃进入网站根目录
cd / var / www / html /
＃在网站根目录中创建index.php，
写入phpinfo函数，测试网站php是否正常
echo ‘<？php phpinfo（）;？>’ > index.php

安装MariaDB的
yum install mariadb-server -y
＃启动mariadb-server
systemctl start mariadb
＃设置mariadb-server开机自启
systemctl enablemariadb
＃ mysql初始化，注意记住mysql密码
mysql_secure_installation
＃ 重启mariadb服务，读取my.cnf新配置
systemctl restart mariadb
＃ 登录数据库，查看字符编码
mysql -uroot -p
输入 \s 查看编码
